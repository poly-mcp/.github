## About PolyMCP

**PolyMCP** is the organization building **PolyMCP** and related tooling for exposing tools, connecting MCP servers, and running agents that orchestrate those tools via the **Model Context Protocol (MCP)**.

We focus on:
- production-ready developer tooling
- consistent interfaces (tools/resources/prompts)
- repeatable patterns for agents and orchestration
- great docs, examples, and reference implementations

## Flagship Project

### PolyMCP

**PolyMCP** is a universal MCP toolkit and agent framework for **Python** and **TypeScript**.

- GitHub: https://github.com/llm-use/polymcp
- Website: https://www.poly-mcp.com
- PyPI: https://pypi.org/project/polymcp/
